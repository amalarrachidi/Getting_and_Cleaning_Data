##Coursera-Getting-and-Cleaning-Data

Those are the steps to follow this R script of this project:

1 - Load the data from https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip and copy it in the repositery named ‘Getting-and-Cleaning-Data’ in your working directory.

2 - Copy the R script ‘run_analysis.R in your working directory.

3 – Source the R code : source("run_analysis.R")

4- lunching the script will generate 2 files : merged_data.txt and data_final.txt
